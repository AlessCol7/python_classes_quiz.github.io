🌲 Capitolul I · Variabile Globale
🏰
Decretul Reginei Zânelor
Variabilele globale trăiesc peste tot
Lecție
🧚 ZÂNA CODIȚA SPUNE:Regina Zânelor a proclamat un decret pentru toată Pădurea Fermecată: „Numele meu este Lumina și puterea mea este 100!" Acest decret — variabila globală — este cunoscut de toată lumea din pădure, de orice funcție-copac și orice casă-funcție. Oricine poate citi acest decret!

✦ Reguli ale Variabilelor Globale

  • Se definesc în afara oricărei funcții — la nivelul principal al programului
  • Sunt accesibile (de citit) din orice funcție
  • Dacă vrei să le modifici dintr-o funcție, trebuie cuvântul magic global
  • Există pe tot parcursul rulării programului
variabile_globale.py
nama_reginei = "Lumina" # variabilă GLOBALĂ 🏰 putere = 100 # variabilă GLOBALĂ 🏰 def saluta_regina(): # Putem CITI variabila globală din funcție: print("Bună ziua, Regina", nama_reginei) print("Puterea ei:", putere) def anunta_padurea(): print(nama_reginei, "conduce pădurea!") saluta_regina() # → Bună ziua, Regina Lumina anunta_padurea() # → Lumina conduce pădurea!

🍄 Capitolul II · Variabile Locale
🏠
Secretele din Căsuța Ciupercii
Variabilele locale trăiesc DOAR înăuntru
Lecție
🧚 ZÂNA CODIȚA:Fiecare funcție-căsuță are propriile ei secrete — variabile locale. Acestea se nasc când funcția pornește și dispar ca fumul de vrajă când funcția se termină. Nimeni din afara căsuței nu le poate vedea sau atinge!

✦ Reguli ale Variabilelor Locale

  • Se definesc înăuntrul unei funcții
  • Există DOAR cât timp funcția rulează
  • Dispar complet când funcția se termină
  • Nu pot fi accesate din afara funcției — NameError!
  • Pot avea același nume ca o variabilă globală — dar sunt diferite!
variabile_locale.py
aroma_padurii = "pin" # GLOBALĂ — toată lumea știe def magia_ciupercii(): secretul = "rețeta de filtru" # LOCALĂ — secret! cristale = 7 # LOCALĂ — secret! print("Înăuntru știu secretul:", secretul) print("Și aroma globală:", aroma_padurii) magia_ciupercii() # Încearcă să accesezi localul din afară: # print(secretul) ← NameError! 💥 Nu există! # print(cristale) ← NameError! 💥 A dispărut!

🌸 Capitolul III · Parametrii Funcției
🪄
Ingredientele Magice ale Vrăjii
Parametri = ce aduci când intri într-o funcție
Lecție
🧚 ZÂNA CODIȚA:Când o zână intră în Camera Vrajilor, aduce cu ea ingrediente speciale — acestea se numesc parametri. Camera (funcția) le primește, le folosește în vrajă, dar ele aparțin doar acelei vizite. Fiecare zână poate aduce ingrediente diferite!
Anatomia funcției cu parametri:
Definiție:
def face_filtru
( planta, cantitate )
:
Apel:
face_filtru
( "levănțică", 3 )
→ planta="levănțică", cantitate=3
🌸 Parametru = variabila din definiție  |  💎 Argument = valoarea dată la apel
parametri_functii.py
def face_filtru(planta, cantitate): # planta și cantitate = PARAMETRI print(f"🌿 Filtru din {planta}, {cantitate} lingurițe") print("✨ Vraja este gata!") # Apelăm cu argumente diferite: face_filtru("lavandă", 2) # → Filtru din lavandă, 2 lingurițe face_filtru("mușețel", 5) # → Filtru din mușețel, 5 lingurițe face_filtru("mentă", 1) # → Filtru din mentă, 1 lingurițe
💡
Parametrii sunt Variabile Locale! Parametrii (planta, cantitate) se comportă exact ca variabile locale — există doar în interiorul funcției, primesc valorile argumentelor și dispar când funcția se termină.

💜 Capitolul IV · Parametri cu Valoare Implicită
⭐
Ingrediente cu Cantitate Prestabilită
Default parameters — valori standard
Lecție
🧚 ZÂNA CODIȚA:Uneori camera vrăjilor are deja pregătite cantitățile obișnuite de ingrediente! Dacă zâna nu specifică altceva, se folosesc valorile prestabilite. Acestea se numesc parametri cu valoare implicită (default parameters)!
default_parameters.py
def face_vraja(tip, putere=10, durata="1 oră"): print(f"Vraja {tip}: putere {putere}, durată {durata}") face_vraja("lumină") # → putere 10, durată 1 oră (default!) face_vraja("foc", 50) # → putere 50, durată 1 oră (doar putere schimbată) face_vraja("gheață", 80, "3 ore") # → putere 80, durată 3 ore # Keyword arguments (în orice ordine!): face_vraja("vânt", durata="30 min", putere=25)
⚠️
Regula Ordinii! Parametrii cu valori implicite trebuie să vină DUPĂ parametrii fără valori implicite! def f(a, b=5) ✅ corect. def f(b=5, a) ❌ eroare de sintaxă!
